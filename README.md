## <a name="introduction">ğŸ¤– Introduction</a>

Built with the latest **Next.js** and **TypeScript**, this project is a modern, responsive replica of the popular video conferencing tool, Zoom. It enables users to securely log in, create meetings, and access various meeting functionalities such as recording, screen sharing, and managing participants.

This project showcases professional full-stack development skills, integrating real-time video communication with a sleek, user-friendly interface.

## <a name="quick-start">ğŸ¤¸ Quick Start</a>

Follow these steps to set up the project locally on your machine.

**Prerequisites**

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

**Cloning the Repository**

```bash
git clone https://github.com/Ujjwal120605/yoom.git
cd yoom
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```

**Set Up Environment Variables**

Create a new file named `.env` in the root of your project and add the required credentials as shown in `.env.example` (you will need keys from Clerk and GetStream).

**Running the Project**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the project.
